## ğŸ—„ï¸ Database Schema

### Users Table
- id, name, email, phone, password (hashed)
- role (rider/driver/admin)
- is_active, is_verified
- profile_picture, timestamps

### DriverProfile Table
- user_id (FK), license_number
- vehicle_type, vehicle_model, vehicle_plate
- rating, total_rides, is_available
- current_lat, current_lng

### Ride Table
- rider_id (FK), driver_id (FK)
- pickup/destination (address, lat, lng)
- status, vehicle_type
- distance_km, duration_minutes
- estimated_fare, final_fare
- rating, feedback, timestamps

### City Table
- name, state, country
- lat, lng, is_active

### IntercityRide Table
- rider_id, driver_id
- origin_city_id (FK), destination_city_id (FK)
- pickup_address, dropoff_address
- scheduled_date, status
- distance_km, price, passengers

### Vacation Table
- user_id (FK), destination
- hotel_name, hotel_address
- start_date, end_date, total_price
- ride_included, hotel_included
- vehicle_type, passengers, status
- booking_reference

### LoyaltyPoints Table
- user_id (FK), total_points
- tier (bronze/silver/gold/platinum)

## ğŸ”Œ API Endpoints Summary

### Authentication
- `POST /api/auth/register` - Register user
- `POST /api/auth/login` - Login
- `POST /api/auth/driver/register` - Register driver

### Rides
- `POST /api/rides/` - Create ride
- `GET /api/rides/` - Get user rides
- `GET /api/rides/available` - Available rides (drivers)
- `GET /api/rides/{id}` - Get specific ride
- `PATCH /api/rides/{id}` - Update ride
- `DELETE /api/rides/{id}` - Cancel ride
- `POST /api/rides/{id}/rate` - Rate ride

### Users
- `GET /api/users/me` - Current user info
- `GET /api/users/drivers` - List drivers
- `PATCH /api/users/driver/location` - Update location
- `PATCH /api/users/driver/availability` - Toggle availability

### Admin
- `GET /api/admin/stats` - Platform statistics
- `GET /api/admin/users` - All users
- `PATCH /api/admin/users/{id}/toggle-active` - Toggle user
- `DELETE /api/admin/users/{id}` - Delete user

### Intercity
- `GET /api/intercity/cities` - List cities
- `POST /api/intercity/cities` - Create city
- `POST /api/intercity/rides` - Book intercity ride
- `GET /api/intercity/rides` - User's intercity rides
- `PATCH /api/intercity/rides/{id}/accept` - Accept ride

### Vacation
- `POST /api/vacation/` - Create vacation
- `GET /api/vacation/` - List vacations
- `GET /api/vacation/{id}` - Get vacation details
- `DELETE /api/vacation/{id}` - Cancel vacation
- `GET /api/vacation/loyalty/points` - Loyalty points
